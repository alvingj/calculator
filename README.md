# calculator(表达式计算器)
10多年前的实验项目, 目前为j2se实现,直接运行com.ccniit.calculator.Main便可执行该项目, 后续可改造为web实现.

---
## 相关知识
**数据结构** 

自己实现堆栈数据结构com.ccniit.calculator.expValTool.StatcImpOfArray

**配置文件读取与解析**

将运算符的信息放到配置文件中


**反射**

使用反射获取Math类的信息, 并调用Math类的方法进行计算


**面向对象设计**

无论是自己实现的堆栈, 还是对运算符的封装, 还是对表达式进行处理的类都用到了oop
